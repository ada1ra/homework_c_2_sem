# Построение ДКА по регулярному выражению


## Регулярное выражение:  
`digit+(.digit+)?(E(+|-)?digit+)?`


## Алфавит
- `d` – цифра (0-9)
- `.` – точка
- `E` – буква E (латинская, заглавная)
- `+` – знак плюс
- `-` – знак минус


## Состояния

- `0` – начальное
    - ещё не прочитано ни одной цифры
    - это промежуточное состояние
    - возможен переход только по цифре (в состояние 1)
    - в противном случае – ошибка

- `1` – целая часть
    - прочитана хотя бы одна цифра целой части
    - это допустимое состояние
    - возможен переход в:
        - прочитать ещё цифру (остаться в 1)
        - прочитать точку (перейти в состояние 2)
        - прочитать E (перейти в состояние 4)
    - любой другой символ – ошибка

- `2` – ожидание дробной части
    - прочитана точка, ожидается хотя бы одна цифра дробной части
    - это промежуточное состояние
    - возможен переход только по цифре (в состояние 3)
    - в противном случае – ошибка

- `3` – дробная часть
    - прочитана хотя бы одна цифра дробной части
    - это допустимое состояние
    - возможен переход в:
        - прочитать ещё цифру (остаться в 3)
        - прочитать E (перейти в состояние 4)
    - любой другой символ – ошибка

- `4` – ожидание экспоненты
    - прочитана буква E, ожидается знак экспоненты или цифра экспоненты
    - возможен переход в:
        - по цифре (в состояние 6)
        - по + или - (в состояние 5)
    - любой другой символ – ошибка

- `5` – ожидание цифр экспоненты
    - прочитан знак экспоненты (+ или -), ожидается хотя бы одна цифра экспоненты
    - возможен переход только по цифре (в состояние 6)
    - в противном случае – ошибка

- `6` – число экспоненты
    - прочитана хотя бы одна цифра экспоненты
    - это допустимое состояние
    - возможно ввести только цифру (остаться в 6)
    - любой другой символ – ошибка


## Таблица переходов

| Состояние | digit | . | E | + | - |
|-----------|-------|---|---|---|---|
| 0         | 1     | - | - | - | - |
| 1         | 1     | 2 | 4 | - | - |
| 2         | 3     | - | - | - | - |
| 3         | 3     | - | 4 | - | - |
| 4         | 6     | - | - | 5 | 5 |
| 5         | 6     | - | - | - | - |
| 6         | 6     | - | - | - | - |

Прочерк — переход в состояние ошибки


## Примеры работы автомата

### `123`
→ 0-(d)→1 (допуск)

### `123.45`
→ 0-(d)→1-(d)→1-(d)→1-(.)→2-(d)→3-(d)→3 (допуск)

### `12E+34`
→ 0-(d)→1-(d)→1-(E)→4-(+)→5-(d)→6-(d)→6 (допуск)

### `123.E5`
→ 0-(d)→1-(d)→1-(d)→1-(.)→2-(E)→ошибка (после точки нет цифр)

### `E10`
→ 0-(E)→ошибка (нет целой части)
